# GPS Plus v6.4.5

Mobil cihazlarınız için gelişmiş konum belirleme ve GPS yönetim uygulaması.

## 🚀 Özellikler

- **Hassas Konum Belirleme:** Geolocation API kullanarak yüksek doğrulukta konum verisi.
- **Harita Entegrasyonu:** Leaflet ve React-Leaflet ile interaktif harita desteği.
- **Koordinat Dönüşümü:** Proj4js ile farklı koordinat sistemleri arasında dönüşüm.
- **Veri Aktarımı:** Kayıtlı konumları Excel (XLSX), KML ve TXT formatlarında dışa aktarma.
- **Stakeout (Aplikasyon):** Belirli koordinatlara yönlendirme ve mesafe takibi.
- **Geoid Desteği:** EGM96 ve TG20 geoid modelleri ile hassas yükseklik hesaplama.
- **Modern Arayüz:** Tailwind CSS ve Motion (Framer Motion) ile akıcı ve kullanıcı dostu tasarım.

## 🛠️ Teknoloji Yığını

- **Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite 6](https://vitejs.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Maps:** [Leaflet](https://leafletjs.com/)
- **Icons:** [Lucide React](https://lucide.dev/) & [FontAwesome](https://fontawesome.com/)
- **Data Handling:** [SheetJS (XLSX)](https://sheetjs.com/), [JSZip](https://stuk.github.io/jszip/)
- **Geodesy:** [Proj4js](http://proj4js.org/)

## 📦 Kurulum

Projeyi yerel ortamınızda çalıştırmak için:

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

3. Üretim için derleme yapın:
   ```bash
   npm run build
   ```
